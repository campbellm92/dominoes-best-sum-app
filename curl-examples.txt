Valid:

curl -X POST http://localhost:8080/best-chain \
  -H "Content-Type: application/json" \
  -d '{
        "tiles": [
          { "left": 1, "right": 3 },
          { "left": 3, "right": 4 },
          { "left": 4, "right": 6 }
        ]
      }'

Permutation:

curl -X POST http://localhost:8080/best-chain \
  -H "Content-Type: application/json" \
  -d '{
        "tiles": [
          { "left": 1, "right": 3 },
          { "left": 4, "right": 6 },
          { "left": 3, "right": 4 }
        ]
      }'


Invalid - 0:

curl -X POST http://localhost:8080/best-chain \
  -H "Content-Type: application/json" \
  -d '{
        "tiles": [
          { "left": 0, "right": 1 },
          { "left": 1, "right": 4 }
          
        ]
      }'


Invalid - same values:

curl -X POST http://localhost:8080/best-chain \
  -H "Content-Type: application/json" \
  -d '{
        "tiles": [
          { "left": 4, "right": 4 },
          { "left": 3, "right": 4 }          
        ]
      }'


Longer: 

curl -X POST http://localhost:8080/best-chain \
  -H "Content-Type: application/json" \
  -d '{
        "tiles": [
          { "left": 1, "right": 3 },
          { "left": 3, "right": 5 },
          { "left": 5, "right": 8 },
          { "left": 8, "right": 9 },
          { "left": 9, "right": 10 },
          { "left": 10, "right": 2 },
          { "left": 2, "right": 7 },
          { "left": 7, "right": 4 },
          { "left": 4, "right": 6 }
        ]
      }'
